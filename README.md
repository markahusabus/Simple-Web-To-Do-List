# Group To-Do List - Open Source Project

A simple collaborative To-Do List web application built by 18 group members.

## How to Run
1. Clone or download the repository
2. Open `index.html` in any web browser
3. No installation needed!

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Clear all tasks
- Tasks are saved in browser
